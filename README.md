# ShilpaKala Android App

ShilpaKala is a Kotlin + Jetpack Compose Android project for helping Karnataka artisans create branded product photos.

See `PROJECT_REPORT.md` for the full documentation, architecture, and setup notes.

## Quick Features

- Email/password auth and Google Sign-In flow
- CameraX smart camera with product guide
- Bitmap enhancement and brand overlays
- Gemini heritage label generation with offline fallback
- Room database and Firestore sync hooks
- Gallery, sharing, QR code, reminders, and analytics

## Run

Open the folder in Android Studio, select the Embedded JDK 17, sync Gradle, and run on a device/emulator.
